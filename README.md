schemax
=======

SchemaX, a better Schema.org documentation

[JSON format schemas](http://schema.rdfs.org/all.json)


Dependencies
------------

- [JSDuck](https://github.com/senchalabs/jsduck)
- NodeJS



Running Parser
--------------
`node parser.js`


Running JSDuck
--------------

`jsduck --config=config.json`
